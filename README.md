# 🍕 Pizza Order Management System (WinForms)

A robust, user-centric desktop application built with **C#** and **.NET Framework** for streamlining pizza ordering processes. This project demonstrates a solid understanding of **Event-Driven Programming**, **UI/UX design principles**, and **Object-Oriented Logic**.

---

## 🚀 Key Features

* **Interactive Dashboard:** A clean entry point with intuitive navigation to 'About' and 'Order' modules.
* **Dynamic Pricing Engine:** Real-time price calculation based on size, crust type, toppings, and dining preferences.
* **Comprehensive Order Logic:**
    * Selection-driven price updates using Radio Buttons & Checkboxes.
    * State management with a 'Reset' functionality to clear forms and re-initialize totals.
* **Order Lifecycle Management:** Includes a confirmation stage and a final **Order Summary** generating a clear breakdown of the customer's choices.

## 🛠 Tech Stack & Concepts Applied

* **Language:** C#
* **Framework:** .NET Framework (WinForms)
* **Architecture:** Modular Form Design (separation of concerns across multiple windows).
* **Logic:** Implementation of conditional pricing algorithms and UI state synchronization.
